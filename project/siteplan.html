Anna Thurston Website Plan

Site Name

Harmony Hub

Site Purpose

The site provides a hub for local aspiring musicians of all ages by offering beginning to intermediate piano, voice, and guitar lessons. It serves as a community resource and learning center for developing musical skills.

Scenarios

  • What is proper positioning and technique for my instrument?
  • Where can I learn to improve my music theory knowledge and musicianship skills?

Color Scheme

  • Background Color: #F5F5F5
  • Text Color: #2E2E2E
  • Primary Color: #4A6E78
  • Call-to-Action Color: #417D7A
  • CTA Hover Color: #356B68
  • Card Background: #ffffff

Typography

Font Family: Arial, sans-serif
To be used in the heading, body, and footer.
